Тестовые страницы для веб-аналитики

На этой странице собраны примеры форм для демонстрации настройки макроцелей в Яндекс Метрике и Яндекс Tag Manager.

Классическая форма с перезагрузкой страницы После отправки страница обновляется, данные передаются через GET-параметры. Форма со страницей благодарности После отправки пользователь переходит на отдельную страницу thank-you.html. Форма без перезагрузки страницы Форма отправляется через JavaScript/AJAX, URL страницы не меняется. Псевдоформа без события submit Внешне похожа на форму, но в коде нет элемента form, поэтому submit не происходит.

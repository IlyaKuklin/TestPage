Примеры форм

Тестовые страницы для демонстрации разных способов отслеживания макроцелей.

Классическая форма с перезагрузкой страницы После отправки страница обновляется, данные передаются через GET-параметры. Форма со страницей благодарности После отправки пользователь переходит на отдельную страницу thank-you.html. Форма без перезагрузки страницы Событие submit происходит, но стандартная отправка отменяется через JavaScript. Псевдоформа без события submit Внешне похожа на форму, но в коде нет элемента form, поэтому submit не происходит. Многошаговая форма Форма из трёх шагов: после каждого шага данные отправляются в dataLayer, submit происходит в конце. Форма внешнего сервиса — GetCourse Форма подключается через внешний скрипт, поэтому её отправка может отслеживаться иначе, чем обычная HTML-форма.

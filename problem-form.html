Проблемная форма

Эту форму можно “отправить” даже с пустыми полями. Сообщение об успехе появляется всегда.

Спасибо! Заявка успешно отправлена.
Проблема: сообщение об успехе появляется всегда, даже если поля пустые. Поэтому отслеживать только видимость этого сообщения нельзя.
Код проверки формы и отправки события в dataLayer
<script>
  window.dataLayer = window.dataLayer || [];

  (function () {
    const form = document.querySelector('#problemForm');
    const button = document.querySelector('#sendButton');
    const nameInput = document.querySelector('#name');
    const phoneInput = document.querySelector('#phone');

    if (!form || !button || !nameInput || !phoneInput) {
      console.warn('Problem form elements not found');
      return;
    }

    button.addEventListener('click', function () {
      const name = nameInput.value.trim();
      const phone = phoneInput.value.trim();

      if (!name || !phone) {
        console.log('Форма не засчитана: не все поля заполнены');
        return;
      }

      window.dataLayer.push({
        event: 'problem_form_valid_success'
      });
    });
  })();
</script>
